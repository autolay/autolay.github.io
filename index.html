Tasks

Given an image (or a sequence of images) \(I\), usually in perspective view, we wish to learn a labeling function \(\Phi\) of all points within an area \(R\) (usually rectangular) around the camera. Here, \(R\) refers to bird's eye view, which is the top-down orthographic view of the ground plane. The labeling function \(\Phi\) must produce a label distribution (over a set of \(N\) classes) for world points within the swath \(R\), regardless of whether or not they are imaged in \(I\). We divide the amodal layout estimation task into three sub-tasks.

  • Road Layout
  • Vehicle Layout
  • Lanes Layout
    • Ego-lane Layout
    • Other lanes Layout

Download

The Autolay dataset contains road, vehicle and lane layouts with the hierarchy same as the kitti RAW dataset. The Autolay train dataset contains layouts for 24 sequences belonging to City, Residential and Road data categories. The test dataset contains layouts for 22 sequences.

The corresponding RGB image for any layout can be found in the image_02 folder of the particular sequence in the RAW dataset. The road and the vehicle layouts are available as .png files, and the lane layouts are available as .npy files containing semantic labels for each lane. More information regarding the semantic labels can be found in this readme file.
